## Overview
This repository contains the Android mobile application developed for milk collection drivers in the **MilkFlow** system.

The Driver Application allows drivers to view assigned milk collection routes, access farmer locations, and follow optimized collection plans generated by the VRP service. The application interacts with the central backend to receive real-time route and logistics information.

## Key Features
- View assigned routes
- Farmer location access
- Route guidance support
- Backend API integration

## Technology Stack
- Android (Java / Kotlin)
- REST APIs

## Academic Context
This project is developed as part of the **Bachelor of Computer Science (BCS)** undergraduate degree program at the **University of Ruhuna, Sri Lanka**.
