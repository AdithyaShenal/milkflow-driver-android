## Overview
This repository contains the **Driver Mobile Application** of the **MilkFlow** system, developed using **React 19**, **Vite**, and **Capacitor**.

The application is designed as a **lightweight, high-performance mobile app** for milk collection drivers. It allows drivers to view assigned collection routes, access farmer locations, and follow optimized routing plans generated by the VRP optimization service.

By leveraging a modern web-based mobile architecture, the application ensures fast load times, low resource usage, and smooth user experience on Android devices.

## System Context
The Driver Mobile Application is **one component of the larger MilkFlow system**, a comprehensive dairy supply chain management platform.  
Other major components include:
- Admin Web Dashboard (React)
- Main Backend (Node.js)
- VRP Optimization Service (Spring Boot)
- Farmer Mobile Application (React + Capacitor)

## Key Features
- View assigned milk collection routes
- Access farmer location details
- Integration with VRP-optimized route data
- Lightweight and fast mobile performance
- Backend API integration

## Technology Stack
- React 19
- Vite
- Capacitor
- REST APIs
